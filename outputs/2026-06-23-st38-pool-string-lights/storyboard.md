## 设计解析

- 景别逻辑：第一镜用看球小全景和足球起跳建立世界杯主题；第二镜用足球近景和水面反射做转场线索；第三镜用人物中近景和派对小全景证明氛围已改变但继续藏产品；第四镜用足球高光到 ST38 灯泡的特写 match cut 完成反转和产品记忆。
- 运镜逻辑：全片围绕足球运动轨迹设计：脚尖挑球、空中旋转、水面倒影、人物接球、再次抛球、圆形高光切成灯泡。镜头运动从足球出发，最后自然滑到灯串，不像硬插产品。
- 物理世界连续性：时间始终是夏季夜晚，空间始终是同一处后院泳池看球派对；足球、泳池、餐桌、投影幕、情侣和朋友连续存在；前三镜不露产品，但暖色圆点与第四镜揭示的灯串位置和反光方向保持因果一致。
- 分镜连续性：进球带来足球起跳，足球旋转产生圆形暖色高光，暖色高光在水面和人群中形成悬念，最终高光丝滑反转成 ST38 灯泡，解释整场暖光来源。
- 情绪曲线：进球狂欢 -> 足球慢动作悬念 -> 后院像球场一样亮起来 -> ST38 灯串揭晓，完成世界杯主题到灯串产品的丝滑反转。
- 品牌记忆点：记忆落在“足球圆形高光 match cut 到 ST38 灯泡 + 泳池整圈暖光覆盖 + 太阳能板 + 遥控器”上，产品最后登场但直接解释前面全部氛围变化。
