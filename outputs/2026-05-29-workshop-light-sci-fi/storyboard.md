# 双管车间灯绝地武士版 15 秒分镜

## 故事核心

黑暗车间被拍成星际决斗场，两名绝地武士式角色用“双刃光剑”高速交锋，高潮处镜头反转揭示光剑本体其实是黑灰端盖双管 LED 车间灯，最后它从武器错觉回到工位上方，用均匀白光点亮整条工作台。

## 创意方向

- 产品 / 品牌：双管长条 LED 车间灯
- 目标受众：车库、维修间、车间、仓库、工作台照明用户
- 核心卖点：双管发光、长条覆盖、白光均匀、黑灰端盖识别度、适合工位上方安装
- 视频风格：绝地武士式科幻对决 + 双刃光剑误导 + 产品真实用途反转
- 情绪曲线：神秘登场 -> 激烈对决 -> 真相反转 -> 工位点亮 -> 产品记忆
- 场景设定：夜间车间被灯光拍成科幻决斗场；真实空间仍是金属工位、工具墙、货架、墙面开关和上方安装位

## 设计解析

- 景别逻辑：前两镜必须明确出现绝地武士式角色和决斗动作，第三镜用产品端盖与双管发光面完成反转，第四镜展示安装和照明结果，第五镜锁定产品外形。
- 运镜逻辑：低机位建立神秘登场，环绕强化对决，贴近端盖制造反转，拉远把武器错觉转成照明工具，推近完成产品记忆。
- 物理世界连续性：全片发生在同一夜间车间；人物始终是两名深色长袍斗篷的绝地武士式角色；产品始终是黑灰端盖、双管并排的长条 LED 车间灯；“光剑”错觉来自双灯管发光、烟雾、金属反光和武士持握动作。
- 分镜连续性：先把双管灯当作双刃光剑参与决斗，再用端盖和灯管细节揭示真实身份，最后把它安装到工位上方完成照明。
- 情绪曲线：神秘、激烈、反转、释放、记忆。
- 品牌记忆点：观众先记住“双刃光剑”，再发现它对应产品的“双管长条形态”，从科幻梗自然落到车间照明卖点。
