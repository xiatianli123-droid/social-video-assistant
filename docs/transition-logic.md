# 转场逻辑参考

本文档用于短视频分镜里的无缝转场、空镜转场和动势匹配设计。写转场时不要只写“丝滑”“无缝”或“镜头一转”，必须说明转场类型、共享视觉锚点和叙事目的。

## 总原则

- 先确定转场服务的叙事目的：承接动作、压缩时间、切换空间、表达情绪、突出产品记忆，或制造前后反差。
- 再选择视觉锚点：运动方向、遮挡物、形状、颜色、光线、构图、动作、产品状态、人物视线或环境声音。
- 最后检查物理可读性：观众能理解为什么从前一镜切到后一镜，或者能明确感知这是一次有意的跳转。

## 转场类型

### 遮挡转场

用前景遮挡、人物经过、布料/手臂/墙面边缘、近距离物体、镜头甩动或同方向运动遮住剪辑点。

- 适用场景：同一空间内压缩动作，或从一个物体遮挡切到另一个空间的同类遮挡。
- 关键锚点：遮挡面积、运动方向、运动速度、画面亮暗变化。
- 写法要求：说明“什么遮住画面”和“遮挡消失后露出什么”。

### 动势匹配转场 / Match Cut

用两个镜头之间相似的运动、形状、角度或动作完成承接。

- 按运动方向区分：下坠、上升、推进、后撤、横移、旋转。
- 按画面内容区分：形状相似、主体相似、颜色相似、光斑相似、产品轮廓相似。
- 按动作区分：抬手接续、转身接续、按键接续、物体落点接续、光线亮起接续。
- 写法要求：说明前一镜的结束锚点和后一镜的开始锚点，例如“足球高光圆形在画面中心旋转，切到同尺寸同位置的 ST38 灯泡发光”。

### 前后承接转场

后一镜直接承接前一镜的动作、物体、构图、颜色状态或信息状态，让观众感觉因果连续。

- 适用场景：展示操作结果、产品状态变化、人物反应、空间从局部到整体的展开。
- 关键锚点：同一物体、同一动作、同一视线方向、同一光色、同一构图中心。
- 写法要求：说明前一镜触发了什么，后一镜展示了什么结果。

### 无缝转场

无缝转场不是单一技巧，而是通过相似画面或隐藏剪辑点，让切换感觉连续。

- 溶解转场 / Dissolve Cut：两个镜头通过相近纹理、色块、光斑、轮廓、产品形状或环境质感交叉融合。
- 隐藏剪辑 / Invisible Cut：剪辑点藏在阴影、黑场、白场、遮挡、快速运动模糊、近距离物体或同色画面里。
- 写法要求：说明采用溶解还是隐藏剪辑，并写清用于隐藏或融合的视觉条件。

### 空镜转场 / Cut Away

在两个相关性较弱的场景之间插入一个环境、物件、天空、光影、街景、室内细节或产品局部镜头，用来表达情绪、时间流逝、地点转换或人物内心状态。

- 适用场景：段落换气、从人物行为切到产品记忆、从室内切到室外、从问题切到解决方案。
- 关键锚点：情绪、光线、时间、声音、地点标识、产品局部或环境细节。
- 写法要求：空镜必须和前后镜头至少有一个明确关系，不能只是好看的插画式镜头。

## 分镜写法检查

每个转场至少回答三件事：

1. 前一镜用什么结束？
2. 后一镜用什么开始？
3. 两者靠什么锚点连接，为什么观众能读懂？

推荐句式：

```text
转场逻辑：用 [前一镜结束锚点] 匹配 [后一镜开始锚点]，通过 [运动/形状/遮挡/光色/动作] 完成 [承接动作/切换空间/压缩时间/表达情绪/突出产品记忆]。
```

示例：

```text
转场逻辑：用足球高光的圆形旋转匹配 ST38 灯泡的圆形暖光，画面中心位置和旋转动势保持一致，完成世界杯热度到庭院灯串的 match cut。
```
