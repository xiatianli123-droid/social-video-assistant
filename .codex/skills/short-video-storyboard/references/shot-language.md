# 镜头语言参考

## 景别选择

| 景别 | 使用场景 | 情绪效果 |
|---|---|---|
| 远景 | 开场、环境、尺度感 | 建立世界、留出氛围 |
| 全景 | 人物和环境关系 | 完整、孤独、融入或对比 |
| 中景 | 人物动作、产品使用 | 自然叙事、推动信息 |
| 近景 | 表情、手部动作、产品细节 | 情感聚焦、体验感 |
| 特写 | 情绪高潮、关键卖点、产品或品牌记忆 | 冲击、记忆、强调 |

常用节奏：远/全景同时建立场景和需求 -> 中景直接进入触发点 -> 近景呈现结果/反应 -> 特写或产品镜头形成记忆。

## 运镜选择

| 运镜 | 适用场景 | 效果 |
|---|---|---|
| 固定镜头 | 建立场景、产品陈列、安静时刻 | 稳定、可信、清楚 |
| 推镜头 | 从整体到局部、强调卖点 | 注意力集中、期待增强 |
| 拉镜头 | 揭示环境、释放情绪 | 空间打开、结果显现 |
| 摇镜头 | 横向扫视空间或对比前后 | 引导视线、展示范围 |
| 移镜头 | 跟随主体穿越空间 | 临场感、流动感 |
| 跟拍 | 人物行动、体验过程 | 同步感、参与感 |
| POV | 让观众代入使用者 | 直接体验、沉浸 |
| 360°环绕 | 展示产品或人物中心感 | 聚焦、掌控、高级感 |
| 航拍 | 场景规模、户外开场 | 开阔、震撼、品牌气势 |
| 慢动作 | 情绪瞬间、质感展示 | 戏剧化、仪式感 |
| 快切 | 多功能、多场景、多情绪 | 节奏、能量、信息密度 |

## 运镜逻辑规则

运镜必须先回答“为什么动”，再回答“怎么动”。每个分镜都写一句运镜逻辑，说明该运动在叙事中的功能。

| 叙事目的 | 首选运镜 | 逻辑写法 |
|---|---|---|
| 建立空间与秩序 | 固定、短促摇镜、航拍 | 在看清环境的同时让痛点、产品或卖点线索进入画面 |
| 揭示隐藏信息 | 摇镜、拉镜、升降、推后拉开 | 随着镜头移动逐步扩大信息量 |
| 聚焦核心卖点 | 推镜头、微距推进、变焦谨慎使用 | 把注意力从空间压缩到产品/细节 |
| 跟随使用过程 | 跟拍、移镜、稳定器运动 | 只跟随最有信息增量的动作结果，避免完整走位拖慢节奏 |
| 展示质感覆盖 | 平移、滑轨、横向移镜 | 让光线、材质或功能在空间中连续显现 |
| 制造情绪变化 | 慢推、慢拉、手持、快切 | 速度和稳定度决定紧张、释放或真实感 |
| 形成产品/品牌记忆 | 固定特写、拉开、环绕 | 把产品、结果画面或品牌元素锁在同一记忆点 |

常用判断：

- 主体不动、信息要稳定：优先固定镜头。
- 主体移动、观众要跟上：优先跟拍或移镜，但短视频中只保留触发点和结果瞬间。
- 卖点藏在细节里：优先推镜头或焦点转移。
- 卖点是空间整体变化：优先拉镜头、摇镜或横向移镜。
- 情绪要从压抑到释放：先固定/慢推，再拉开或升起。
- 高级感要克制：用慢速、稳定、短距离运动，避免无意义甩镜和频繁变焦。

## 画面组织原则

不要在最终分镜表里单独输出“构图”列；画面组织应自然写进 `画面描述` 和 `运镜逻辑`。

| 画面组织 | 使用场景 | 效果 |
|---|---|---|
| 九宫格 | 人物、产品与环境共同叙事 | 自然、耐看 |
| 中央构图 | 产品正面、品牌元素、情绪特写 | 明确、强记忆 |
| 对称构图 | 高级感、专业感、仪式感 | 秩序、平衡 |
| 框式构图 | 门窗、屏幕、前景遮挡 | 层次、聚焦 |
| 视线空间 | 人物看向或走向某处 | 画面顺畅、有呼吸 |

## 常用情绪曲线

| 阶段 | 时长 | 目标 | 常见画面 |
|---|---:|---|---|
| Hook | 2-4s | 抓注意力 | 痛点、强视觉、反差、场景压迫 |
| Build | 2-3s | 直接触发卖点 | 人物已经进入使用状态，问题被产品动作打断 |
| Experience | 4-6s | 证明卖点 | 使用过程、细节特写、感官反馈 |
| Payoff | 按节奏分配 | 释放和记忆 | 结果画面、产品定格、情绪收口、品牌元素 |

## 文案节奏

- 短镜头：一句极短句或一句短角色对白，避免用旁白填满画面。
- 15 秒全片通常 3 个镜头、最多 3 句主文案，避免旁白压住画面。
- 如果用户明确提供 CTA，可放最后 2-3 秒，优先短句：立即体验 / 开启清凉 / 让创意上场。

## 短视频节奏检查

- 每个镜头前 0.5 秒要看出广告功能。
- 人物走位、看见对方、慢慢靠近不能成为镜头主体；这些过程要用剪辑跳跃压缩。
- 首帧优先选择已经发生的触发点或结果瞬间，例如风已经吹动发丝、饮品已经递到手边、产品已经成为主体。
