# 生活方式产品分镜经验

这个文件沉淀生活方式产品短视频分镜的抽象经验。不要在这里保存完整项目分镜、具体产品台词、固定人物关系或不可迁移的场景装饰。

## 使用方式

- 用它判断生活方式产品如何把卖点放进真实场景动作里。
- 只复用结构、节奏、证明方式和误用提醒。
- 不复用具体场景、具体道具、人物关系、台词或产品类别。
- 如果新项目产生可复用经验，优先合并到 `可迁移设计要点` 或 `避免误用`。

## 维护规则

- 主文件只保留抽象经验，不追加完整项目案例。
- 如果出现新的生活方式结构，只新增短经验块，不写完整分镜。
- 新增短经验块只写：适用条件、核心结构、卖点证明方式、容易误用的点。
- 同类结构最多保留 3 条短经验；重复经验合并。
- 连续出现 3 次的经验提炼为规则，再删除重复描述。
- 任何内容如果不能指导下一个不同产品的分镜，就删除或归档。
- 历史完整案例如需保留，移入 `references/archive/lifestyle-cases-YYYY.md`；归档不默认读取。

## 适用条件

- 产品需要通过真实生活场景证明体验，而不是只展示外观。
- 卖点需要人物反应、环境变化、道具反馈或前后状态对比来变得可信。
- 场景包含人、物、空间和时间进程，且这些元素能自然承载产品因果。
- 视频目标是建立使用向往、舒适结果、信任感、购买理由或品牌调性。

## 核心结构

生活方式产品分镜优先使用四段式广告因果：

1. 触发前需求：场景中已经出现问题、缺口、阻力或未满足的体验。
2. 产品介入瞬间：产品以可见动作、状态变化或使用行为进入因果链。
3. 结果证明：人物反应、环境反馈或道具变化证明卖点已经发生。
4. 产品记忆：把产品、结果和场景情绪绑定，形成结尾识别点。

短片不必完整展示过程。优先写触发点和结果瞬间，用剪辑跳跃压缩真实动作。

## 可迁移设计要点

- 场景建立：第一镜同时建立空间、人物关系、产品位置和体验缺口；不要只铺氛围。
- 首帧动作：每个镜头开头就要有信息增量，人物、道具、环境或产品状态已经在变化。
- 卖点证明：用画面里已经存在的元素证明卖点，例如人物表情、正常交流、停留选择、动作变轻松、道具状态变化、空间明暗/温度/秩序改善。
- 产品在场：产品最好从前段就建立位置或存在感，后续结果才有因果来源。
- 反应证明：人物反应要具体到行为变化，不只写“满意”“开心”“舒适”。
- 道具证明：道具必须来自场景常识或前镜建立，不能为了证明卖点临时凭空出现。
- 声音证明：低噪、安静、柔和类卖点用正常音量对话、环境底噪或动作声侧面证明，不写无来源产品声。
- 结尾记忆：结尾必须把产品和用户结果绑定，可以用产品定格、前后状态对比、人物继续停留、空间恢复使用、情绪收束或品牌调性完成。
- 视觉连续：天气、时间、空间、人物、服装、道具、产品状态、光线和色彩系统要贯穿全片。
- 文案克制：旁白或对话只补充画面难以表达的因果，不替代卖点证明。

## 常见结构模式

### 痛点到结果

- 适用条件：卖点能直接解决一个可见痛点。
- 核心结构：问题正在发生 -> 产品触发 -> 痛点减轻或消失 -> 用户继续使用场景。
- 证明方式：人物动作变轻松、停留时间变长、原本失败的活动恢复。
- 容易误用：只写“前后对比”但没有中间的产品因果。

### 场景流失到回归

- 适用条件：产品能让用户重新愿意留在某个空间或活动里。
- 核心结构：人正在离开/回避/中断 -> 产品介入 -> 人重新进入或继续场景 -> 产品与场景记忆绑定。
- 证明方式：座位重新有人、交流恢复、活动继续、空间使用率提高。
- 容易误用：把人物走位写成主体，而不是写“为什么留下来”的卖点结果。

### 细节发现

- 适用条件：卖点和可见性、精准性、质感、安全感、收纳或定位有关。
- 核心结构：关键细节被忽略/丢失/看不清 -> 产品让细节可见或可控 -> 人物完成原本受阻的动作。
- 证明方式：手部动作、视线确认、物件被找回、错误被避免。
- 容易误用：只拍产品特写，没有建立“细节为什么重要”。

### 氛围升级

- 适用条件：卖点偏情绪、空间质感、社交氛围或品牌调性。
- 核心结构：场景情绪不足或活动缺少停留理由 -> 产品改变环境状态 -> 人物互动变自然 -> 产品成为情绪锚点。
- 证明方式：人物距离变近、交流变顺、空间层次变清楚、活动继续。
- 容易误用：只写“变高级/变温馨”，没有具体行为或环境反馈。

## 避免误用

- 不要把某个成功案例的场景强行套到所有产品。
- 不要复制旧台词；文案必须根据产品、场景、目标情绪和镜头时长重写。
- 不要为了证明卖点新增无来源道具、天气、人物或空间。
- 不要把 `音效配乐` 机械列满；只写当前画面真正需要听到的声音。
- 不要复用旧式长视频节奏：人物完整走位、看见对方、慢慢靠近不能成为镜头主体。
- 不要让人物关系戏压过卖点证明；关系和情绪只服务产品因果。
- 不要把抽象形容词当证明；“舒适、安静、高级、安心”必须落到可见动作或状态变化。
